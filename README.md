# Aplikasi Kasir Toko

Aplikasi Kasir Toko adalah sistem berbasis web yang dibuat menggunakan framework Laravel untuk membantu proses transaksi penjualan di toko.

## Fitur
- Login Admin
- Dashboard
- Manajemen Data Barang
- Transaksi Penjualan
- Laporan Penjualan

## Teknologi
- Laravel
- PHP
- MySQL
- Bootstrap
